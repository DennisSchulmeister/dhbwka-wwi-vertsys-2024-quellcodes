Notebook-as-a-Service: API-Gateway

Das API-Gateway selbst

Server-Checks
Informationen GET /info
Metriken GET /metrics
Health Check GET /health

Katalog-Backend

Server-Checks
Informationen GET /api/catalogue/info
Metriken GET /api/catalogue/metrics
Health Check GET /api/catalogue/health
Gerätetypen
Alle Gerätetypen GET /api/catalogue/device-class
Gerätetyp suchen GET /api/catalogue/device-class?q=smart
Einzelner Gerätetyp GET /api/catalogue/device-class/1
Geräte eines Typs GET /api/catalogue/device-class/1/devices
Geräte
Alle Geräte GET /api/catalogue/device
Gerät suchen GET /api/catalogue/device-class?q=lenovo
Einzelnes Gerät GET /api/catalogue/device/1

Auftrags-Backend

Server-Checks
Informationen GET /api/order/info
Metriken GET /api/order/metrics
Health Check GET /api/order/health
Aufträge
Alle Aufträge GET /api/order/order
Auftrag suchen GET /api/order/order?q=smart
Einzelner Auftrag GET /api/order/order/1
Auftrag anlegen POST /api/order/order
Geräte
Existenz prüfen GET /api/order/device/exists/1
Verfügbarkeit prüfen GET /api/order/device/available/1?startTime=2024-02-10&endTime=2024-02-20
